Realtime Bus Tracker

Clicking the 'See Route 77 Bus Locations' button will allow users to see all active Route 77 buses in Arlington and Cambridge, MA. Outbound buses are represented with orange icons, and inbound buses are represented in green.

This project uses Mapbox GL and led to learning to render lines, use async functions, icon markers as divs in the DOM, and utilize large data objects to view bus locations.

View the repository or check out the project!

Eyes-A-Moving

Once the page is loaded, a pair of fairly creepy eyes will greet the user, and the eyes will follow the mouse wherever it is moved.

This project highlights the use of CSS grid, as well as style changes based on the user's actions.

View the repository or check out the project!

PacMan Factory

With the click of a button, users are able to generate PacMan-like characters and then send them darting to and fro across the screen, chomping merrily away.

This project displays DOM manipulation, edge detection, and timers.

View the repository or check out the project!
